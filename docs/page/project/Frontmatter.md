## 第三步：编辑项目页面布局

### 1.选择菜单  
![在这里插入图片描述](./33960154737b409eaed8d9980242b7c4.png)
### 2.编辑项目
>注意事项

1. 先选择布局容器
2. 组件容器必须放到布局容器里面
3. 新页面需要先拖拽布局容器 通常是表格布局
4. 页面数据 是当前菜单页面的数据 与开发同思想 先定义：变量名，函数，然后才可以绑定函数 以及变量名
5. 绑定变量 1.在页面数据先创建变量才可以选择变量 默认都是v-model 数据双向绑定  
### 3.容器/组件
1. 双击可以查看 容器或者组件所有 属性，激活是边框是实线选中状态
